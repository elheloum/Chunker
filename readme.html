QUELQUES EXPLICATIONS SUR LE PROJET

Le but de ce projet de découpé un texte récupéré en entrée en CHUNK

Qu'est-ce qu'un CHUNK ?

Selon Miller, les chunks sont des unités(blocs) fonctionnelles de contenu informationnel. Ces blocs sont figés à l'intérieur mais sont intermutable entre eux. Pour plus d'explications, vous pouvez consulter l'article de Steven P. Abney.

Source: Site de Thomas LEBARBE

Quelles sont les entrées pour obtenir le texte découpé en chunk?

Tout d'abord, il faut préciser les catégories des mots qu'on trouvera dans notre texte. ATTENTION, il n'est pas nécessaire de donner les catégories de la totalité des mots, il faudra juste indiquer celles qui sont pertinentes et utiles pour faire notre découpage.

Ensuite, il faut écrire les règles. Ces règles permettront d'effectuer le découpage. Les règles doivent être en cohérence avec le texte d'entrée.

Enfin, le texte d'entrée. Le texte doit être sous forme de TOKEN.

Vous trouverez des exemples des 3 entrées nécessaires sur la page Exemple de données à remplir.

Est-ce qu'on peut enregistrer les données?

Oui, il est possible d'enregistrer les données. Sur la page du projet, vous trouverez le bouton SAVE qui vous permettra d'enregistrer votre travail.

Qu'est-ce qu'on peut avoir en plus ?

Si vous êtes curieux et vous souhaitez savoir combien de fois chaque règle a été utilisé, vous pouvez cliquer sur le bouton GRAPHIQUE DE STAT que vous trouverez sur la page du projet. Ce bouton vous donnera un graphique indiquant le nombre de fois d'utilisation de chaque règle.
